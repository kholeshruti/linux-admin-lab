#Linux Basics

## pwd
Shows current working directory
Example: root@ubuntu:~$ pwd

## ls
List files and directories
Example: root@ubuntu:~$ ls test

## cd 
Changes current working directory
Example: root@ubuntu:~$ cd test2

## mkdir
Creates the new directory
Example: root@ubuntu:~$ mkdir mydirectory

## touch
Creates a new empty file if it doest not exist already
Example: root@ubuntu:~/test2$ touch records.txt

## cp
Copy files from one folder to another
Example: root@ubuntu:~/test2$ cp records.txt /root/test

## mv 
Moves file from folder to another/Renames the files

## rm 
Removes the selected file

## rmdir 
Removes the selected directory
