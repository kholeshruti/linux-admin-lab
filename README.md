#Linux Admin Lab

A beginner-friendly Linux project created to learn the fundamental skills required for Cloud Computing and DevOps.

## Project Objective
This project demonstrates practical Linux administration skills including:

- Linux file management
- User management
- File permissions
- Package installation
- Nginx installation and management
- Process management
- Log management
- SSH basics
- Bash scripting

## Topics Covered

### 1. Linux Basics
- pwd
- ls
- cd
- mkdir
- rm
- cp
- mv
- touch

### 2. User Management
- whoami
- useradd
- passwd
- Viewing users

### 3. File Permissions
- ls -l
- chmod
- Permission values (777, 755, 644)

### 4. Package Management
- apt update
- apt install

### 5. Nginx
- Install Nginx
- Start and stop the service
- Restart and reload the service
- Check service status
- Verify using `curl localhost`

### 6. Process Management
- ps aux
- grep
- top
- systemctl

### 7. Logs
- journalctl
- access.log
- error.log

### 8. SSH
- OpenSSH
- SSH service
- hostname -I
- ssh localhost

### 9. Bash Scripting
Created a simple backup script that copies a file into the backups directory.
